# Federal Register Limits

- The API is authoritative for published Federal Register documents, not for social-media reaction or comment sentiment.
- Search results are only as good as the term and structured filters used.
- Publication dates are date-only fields, so this skill maps mission UTC windows to publication dates rather than exact timestamps.
- `excerpts` may contain HTML markup and highlighted match spans.
- This skill does not fetch linked full-text HTML, XML, or PDF bodies.
- Some policy workflows still require Regulations.gov for docket-comment evidence; Federal Register is complementary, not a replacement.
